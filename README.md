# ✈️ ERNAM Digital Twin: Master Documentation

**Status**: In Development  
**Owner**: Solis Digital Group  
**Languages**: French (Primary), English (Secondary)

## Executive Summary
ERNAM Digital Twin is a bilingual, integrated web and mobile application designed to digitize the operations of the **École Régionale de la Navigation Aérienne et de Management**. It creates a "digital twin" of the institution's physical and administrative processes, focusing on three core pillars: **Training, Governance, and Archives**.

### Core Objectives
*   **Centralize Data**: Move from fragmented paper/legacy systems to a unified cloud database.
*   **Streamline Certification**: Manage ICAO/ASECNA training lifecycles and certifications digitally.
*   **Enhance Decision Making**: Provide Directors with real-time governance dashboards.

## Tech Stack (Modern Experiential)
*   **Frontend**: React.js (Web) / React Native (Mobile option)
*   **Styling**: Tailwind CSS + Framer Motion (Cinematic/Aura feel)
*   **Backend**: Supabase (PostgreSQL, Auth, Storage, Edge Functions)
*   **AI**: Agents for "Chat with Archive" and recommendations.
