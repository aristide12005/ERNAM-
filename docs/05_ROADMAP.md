# 05. Development Roadmap

## Phase 1: MVP (The Core)
**Goal**: Digitization of records and basic catalog.
*   [ ] **Setup**: Supabase Project, Auth (Roles: Student, Admin).
*   [ ] **Module - Archives**: Build the "Digital Vault" and digitize old records. Simplest value add.
*   [ ] **Module - Training**: Build the Course Catalog view (Read-only).

## Phase 2: Interaction
**Goal**: Active user engagement.
*   [ ] **Enrollment**: Enable "My Courses" area and enrollment flows.
*   [ ] **Trainer Dashboard**: Allow file uploads and course management.
*   [ ] **Localization**: Implement full French/English toggle.

## Phase 3: Governance & AI (The "Twin")
**Goal**: Advanced features and intelligence.
*   [ ] **Director Dashboard**: Charts, Financial Pulse, Real-time usage.
*   [ ] **AI**: "Tower" Chatbot and AI Search for Archives.
*   [ ] **Mobile App**: React Native release for Trainees.
