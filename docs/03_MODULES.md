# 03. Functional Modules

## Module 1: TRAINING (Formation)
*   **Course Catalog**: Searchable database (Sûreté de l'aviation, Contrôle Aérien).
*   **LMS (Learning Management System)**:
    *   Uploads: PDFs, Videos.
    *   Interface: Quiz/Exam.
    *   Live Sessions: Calendar integration.
    *   **"Hangar" View**: Courses displayed as "Mission Cards" or 3D objects.
*   **Enrollment Flow**: Registration -> Payment Validation -> Access Grant.
*   **Evaluation**: Digital grading sheets and report cards.

## Module 2: GOVERNANCE (Gouvernance)
*   **Dashboard**: KPIs (Active students, Revenue, Completion rates).
*   **Resource Management**: Assignment of "Salle A", "Simulateur 3" to courses. Conflict prevention.
*   **Compliance**: Audit logs for ICAO/ASECNA standards.
*   **Staff Portal**: Leave management, internal comms.

## Module 3: ARCHIVES (Mémoire)
*   **Digital Vault**: Secure storage of past diplomas/certs.
*   **Search Engine**: Advanced filtering (Year, Nationality, License #).
*   **Alumni Network**: Database for networking and re-marketing.

## "Digital Twin" Features
*   **Asset Twin**: Digital log of physical assets (Simulators, Planes). Maintenance tracking.
*   **People Twin**: Unified record of entire lifecycle (Applicant -> Student -> Alumni -> Instructor).
*   **Process Twin**: Digitized workflows. Forms travel digitally for signatures.
*   **AI Concierge ("Tower")**: Chatbot for FAQs ("Where is my class?", "Renew medical cert").
